The initial MCP Description format and specification work originated at Cisco DevNet and
was released as open source through the [`cisco-open`](https://github.com/cisco-open) GitHub
organization.

The `{mcpdesc}` logo, wordmark, and visual designs were created by **Jeanne and Stève
Sfartz**, who grant them to the `{mcpdesc}` project for use as project branding. Branding
assets (logo, wordmark, favicon, and social images) are **not** automatically covered by the
Apache-2.0 license, which applies to code, documentation, and specification contributions.

The `{mcpdesc}` project is an independent open source initiative led by its maintainers; it is
**not a Cisco product, service, or official Cisco standardization effort**.

## The MCP Toolkit

The **MCP Toolkit** tool suite keeps its name and its npm packages
(`@cisco_open/mcptoolkit-*`). It is an open-source project initiated by Cisco DevNet and
maintained at cisco-open, released under the Apache-2.0 license. It is one of the first tool
suites built around `mcpdesc` — not the whole `{mcpdesc}` project. See the
[MCP Toolkit at Cisco Open](https://github.com/orgs/cisco-open/repositories?q=mcptoolkit).

"MCP" and "Model Context Protocol" refer to the open protocol ecosystem. The `{mcpdesc}`
project is not affiliated with or endorsed by Anthropic, or other MCP ecosystem vendors
unless explicitly stated.
